Pac-it-up Github

Legal Text Classification Github

A self proposed final project for my Natural Language Processing class. This project involved looking into several different ways to make state-of-the-art NLP conventions more performant on legal text classification. It taught me a lot about how much work there is to do for legal ML, and it was a lot of fun!
